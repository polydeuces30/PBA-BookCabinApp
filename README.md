# Sentiment Analysis Review BookCabin

Proyek ini bertujuan untuk menganalisis sentimen pengguna terhadap aplikasi BookCabin berdasarkan review di Google Play Store.

## Dataset
Data diambil menggunakan library `google-play-scraper` dengan total hingga 20.000 review.

## Tahapan
1. Scraping data
2. Preprocessing (cleaning, stopword removal, stemming)
3. Labeling sentiment berdasarkan rating
4. Visualisasi WordCloud
5. Modeling menggunakan Naive Bayes
6. Evaluasi model

## Tools
- Python
- Google Colab
- Scikit-learn
- Sastrawi
- WordCloud
